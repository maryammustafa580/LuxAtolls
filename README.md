# Lux Atolls

Lux Atolls is a luxury travel and resort discovery website built with Next.js and TypeScript. It provides a modern, responsive platform for exploring premium destinations, resorts, and vacation experiences.

## Technologies

* Next.js
* React
* TypeScript
* CSS

## Features

* Responsive design
* Modern UI/UX
* Hero section with media background
* Resort showcase
* Quote request form
* Fast and optimized performance

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
